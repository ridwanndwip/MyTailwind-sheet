Cara Install TailwindCSS

Install menggunakan Tailwind-CLI
  • Buka terminal dan tulis perintah
    npm install -D tailwindcss postcss autoprefixer (install tailwind, postcss, autoprefixer via npm. -D adalah dependencies)

  • Masih di terminal, dan tulis perintah
    npm tailwindcss init (digunakan untuk membuat file konfigurasi default untuk Tailwind CSS di proyek Anda, file tersebut bernama tailwind.config.js)

  • Buatlah file input.css, dan tambahkan kode berikut : @tailwind base;
    @tailwind components;
    @tailwind utilities;

  • Buka terminal kembali, dan tulis perintah
    npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
    perintah diatas digunakan untuk membuat live reload/live change pada tailwindcss.

  • Setup menggunakan Tailwind CLI sudah berhasil, sekarang Anda compile output.css Anda ke HTML dan siap digunakan<link href="./output.css" rel="stylesheet">

Menggunakan Tailwind via Play CDN
  • Tambahkan tag script yang sudah di siapkan oleh tailwindcss dalam tag Head di HTML anda <script src="https://cdn.tailwindcss.com">

Responsive TailwindCSS

  • Pertama, pastikan Anda sudah menambahkan tag meta viewport di dalam tag head, seperti dibawah ini
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Breakpoints Minimum Width Output Css
    sm 640px @media (min-width: 640px) {...}
    md 768px @media (min-width:768){...}
    lg 1024px @media (min-width:1024){...}
    xl 1280px @media (min-width:1280){...}
    2xl 1536px @media (min-width:1536){...}
